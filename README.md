# Task Manager App (Flutter + Firebase)

## Features
- Full CRUD with Firestore
- Real-time updates using StreamBuilder
- Nested subtasks
- Dark mode support
- Input validation

## Setup
1. Clone repo
2. Run:
   flutter pub get
3. Configure Firebase:
   flutterfire configure
4. Run app:
   flutter run

## Enhanced Features
1. Dark Mode (system-based)
2. Nested Subtasks with expand/collapse UI

## Known Issues
- No authentication (open Firestore rules)
